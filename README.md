# Java_Projects

This is a basic chat system project where two user interfaces can chat simultaneously. This project was developed by Nithish Marneni during early academic years and is currently maintained to demonstrate core Java networking principles.</br>
The chat application introduces both ends simultaneous response </br>

How to run: </br>
1. Open in Vscode/ other ides
2. Run java server.java
3. Run java client.java
